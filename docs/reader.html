Karel the Robot

Karel is a simple robot living in a simple world. Karel has been used to introduce millions of students to programming at Stanford and beyond. We have two versions of Karel, which introduce students to coding using either a smaller set of a programming language called "Java" or a programming language called "Python". Each version has been translated into multiple human languages.
