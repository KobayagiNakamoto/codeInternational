第9章：额外的Karel功能

画角

卡雷尔还可以使用以下颜色在其世界中达到paint corner秒：

  • 空白 ，即move是正方形上的任何颜色
  • 黑色
  • 蓝色
  • 青色
  • 深灰色
  • 灰色
  • 绿色
  • 浅灰
  • 品红
  • 橙子
  • 粉
  • 红
  • 白色
  • 黄色

要使用Karel的绘画功能，请使用以下命令：

命令 它能做什么
paint_corner(颜色名称) 卡雷尔（Karel）为他所站在的角落着色 颜色名称 。例如，要为角落涂上蓝色，您可以编写 paint_corner(蓝色).
边角颜色是(颜色名称) 检查此问题的条件：Karel站在拐角处的颜色是 颜色 名称 ？
